## Server 폴더 관련사항 정리

# server 구동법

server 폴더가 현재 작업 디렉토리인 채로 yarn dev나 yarn start 명령 실행

# 폴더

* server

정적인 파일들을 담아두는 디렉토리

* v1

ver 1.0.0 파일들을 담아두는 디렉토리

# 파일

* index.ts

서버를 구동한 후 localhost:3000 으로 접속할 시 나오는 기본 페이지

* api.ts

router의 동작을 확인하기 위한 테스트 페이지

* user.ts

params로 받은 id와 일치하는 사용자의 정보를 출력하는 페이지

* test
